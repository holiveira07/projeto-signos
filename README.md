# Projeto Signos

Projeto desenvolvido para a disciplina de Programação Web.

O sistema identifica o signo do usuário com base na data de nascimento.

## Tecnologias utilizadas

- HTML
- CSS
- Bootstrap
- JavaScript
- PHP
- XML

## Como executar

1. Instalar XAMPP
2. Colocar na pasta htdocs
3. Acessar via localhost
